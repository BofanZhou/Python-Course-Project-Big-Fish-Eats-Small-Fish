# 大鱼吃小鱼 (Big Fish Eats Small Fish)

一个使用 Pygame 开发的经典"大鱼吃小鱼"游戏，支持**单人模式**和**双人合作模式**。

## 游戏截图

*游戏界面包含海洋背景、玩家控制的大鱼、随机生成的敌方鱼群、分数和生命值显示等*

## 功能特性

- **单人模式**：使用 WASD 控制大鱼移动
- **双人模式**：玩家1使用 WASD，玩家2使用方向键，合作捕食
- **动态难度**：随着时间推移，敌方鱼的生成速度、数量和体型会逐渐增加
- **多种敌方鱼行为**：直线游动、正弦波游动、之字形游动
- **音效系统**：内置吃鱼音效、受伤音效和游戏结束音效
- **中文界面**：完整的中文菜单、UI 和提示信息
- **胜利条件**：达到 3000 分即可获胜

## 操作说明

| 玩家 | 上移 | 下移 | 左移 | 右移 |
|------|------|------|------|------|
| 玩家1 (WASD) | W | S | A | D |
| 玩家2 (方向键) | ↑ | ↓ | ← | → |

- **ESC**：暂停/继续游戏
- **鼠标点击**：菜单界面选择游戏模式和按钮

## 游戏规则

1. 控制你的鱼在海洋中移动
2. **吃掉比自己小的鱼**来获得分数并成长
3. **躲避比自己大的鱼**，否则会被吃掉并失去一条生命
4. 初始有 **3 条生命**，生命耗尽则游戏结束
5. 任意玩家达到 **3000 分** 即可获胜
6. 鱼的体型越大，移动速度会略微降低

## 安装与运行

### 环境要求

- Python 3.x
- Pygame

### 安装依赖

```bash
pip install pygame
```

### 运行游戏

```bash
python "Bigfffffish eat small fish.py"
```

## 技术亮点

- **面向对象设计**：PlayerFish、EnemyFish、Game、DifficultyManager、SoundManager、UIManager 等类的合理拆分
- **无需外部资源**：所有音效通过代码动态生成，无需额外的音频文件
- **自适应字体**：自动检测系统中文字体（微软雅黑/黑体/宋体），确保中文正常显示
- **状态机模式**：MENU → PLAYING → PAUSED → GAME_OVER/WIN 的清晰状态流转
